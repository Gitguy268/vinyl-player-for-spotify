# Vinyl Player for Spotify

A web-based retro vinyl record player interface for your Spotify playback. 

## Features
* **Visual Vinyl Player**: Enjoy a classic vinyl aesthetic while listening to your favorite Spotify tracks.
* **Web-Based**: Built with standard web technologies.
* **Spotify Integration**: Connects with your Spotify account to display and control playback.

## Getting Started

### Prerequisites
* A modern web browser.
* A Spotify Premium account (required for Web Playback API features).
* Spotify Developer Application credentials (Client ID) if hosting your own instance.

### Setup
1. Clone this repository:
   ```bash
   git clone https://github.com/Gitguy268/vinyl-player-for-spotify.git
   ```
2. Navigate to the project directory.
3. Add your Spotify Client ID to the configuration (if applicable).
4. Serve the directory using a local web server (e.g., Live Server in VS Code, or `python -m http.server`) and open `index.html` in your browser.

## Usage
1. Click to authenticate with your Spotify account.
2. Start playing music and watch the vinyl spin!
